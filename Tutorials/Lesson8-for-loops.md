# Lesson 8
## For loops

A for loop repeats code for each item in a sequence.

---

### Looping through a list

#### Example:

    names = ["Paddy", "Sarah", "John"]

    for name in names:
        print(name)

will display:

    >>> Paddy
    >>> Sarah
    >>> John

---

### Using range()
The range() function is useful when you want to repeat code a certain number of times.

#### Example:

    for number in range(5):
        print(number)

will display:

    >>> 0
    >>> 1
    >>> 2
    >>> 3
    >>> 4
