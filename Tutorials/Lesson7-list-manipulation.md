## List manipulation

A list stores multiple values in one variable.

Lists use square brackets.

#### Example:

    names = ["Paddy", "Sarah", "John"]
    print(names)

will display:

    >>> ['Paddy', 'Sarah', 'John']

---

### Indexing lists
You can get one item from a list using its index.

Python starts counting from 0.

#### Example:

    names = ["Paddy", "Sarah", "John"]
    print(names[0])

will display:

    >>> Paddy

---

### Changing a list
You can add, remove and change items in a list.

#### Example:

    numbers = [1, 2, 3]

    numbers.append(4)
    numbers.remove(2)
    numbers[0] = 10

    print(numbers)

will display:

    >>> [10, 3, 4]
