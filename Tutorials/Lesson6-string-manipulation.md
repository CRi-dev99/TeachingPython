## String manipulation

String manipulation means changing or using text in different ways.

---

### Joining strings
You can join strings together using the + symbol.

#### Example:

    first_name = "Paddy"
    second_name = "Murphy"

    print(first_name + " " + second_name)

will display:

    >>> Paddy Murphy

---

### String methods
Python has useful methods for strings.

#### Example:

    message = "hello"

    print(message.upper())
    print(message.capitalize())

will display:

    >>> HELLO
    >>> Hello

---

### Indexing strings
You can get one character from a string using its index.

Python starts counting from 0.

#### Example:

    word = "Python"
    print(word[0])
    print(word[1])

will display:

    >>> P
    >>> y
